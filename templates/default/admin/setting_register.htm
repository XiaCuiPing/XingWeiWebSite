开放用户注册：
是 否 设置是否允许游客注册成为网站会员。
新用户注册验证:
选择“无”用户可直接注册成功；选择“Email 验证”将向用户注册 Email 发送一封验证邮件以确认邮箱的有效性； 选择“人工审核”将由管理员人工逐个确定是否允许新用户注册
发送欢迎信息：
  • 不发送
  • 发送欢迎短信息
  • 发送欢迎Email
 可选择是否自动向新注册用户发送一条欢迎信息
欢迎邮件标题：
系统发送的欢迎信息的标题，不支持 HTML，不超过 75 字节。
欢迎邮件内容：
系统发送的欢迎信息的内容。标题内容均支持变量替换，可以使用如下变量:
{username} : 用户名
{time} : 发送时间
{sitename} : 站点名称
{adminemail} : 管理员email
显示许可协议：
是 否 新用户注册时显示许可协议
许可协议内容：
注册许可协议的详细内容
