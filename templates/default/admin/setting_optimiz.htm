URL 静态化:
是 否 URL 静态化可以提高搜索引擎抓取，开启本功能需要对 Web 服务器增加相应的 Rewrite 规则，且会轻微增加服务器负担。
标题附加字:
网页标题通常是搜索引擎关注的重点，本附加字设置将出现在标题中论坛名称的后面，如果有多个关键字，建议用 "|"、","(不含引号) 等符号分隔
其他头部信息:
如需在 中添加其他的 HTML 代码，可以使用本设置，否则请留空
启用缓存：
是 否 设置是否启用页面缓存，启用缓存可以减轻服务器负担，提高浏览速度
缓存首页有效期:
设置首页缓存更新的时间，单位为秒，0 为关闭（此处关闭以后，缓存系数将不再起作用），建议设置为 900。
页面缓存时间：
设置页面缓存更新的时间，单位为秒，0 为关闭。请根据实际情况进行调整，建议设置为 900。
页面 Gzip 压缩：
是 否 将页面内容以 gzip 压缩后传输，可以加快传输速度，需 PHP 4.0.4 以上且支持 Zlib 模块才能使用
